Ping application using ICMP request and reply messages

The Ping application implemented in Python uses ICMP (Internet Control Message Protocol) request and reply messages to test the reachability of a host across an IP network. It measures the round-trip time, records packet loss, and provides statistical summaries of the received echo reply packets. The application sends ping requests to a specified host with a payload containing a timestamp and waits for a reply within one second. If no reply is received, it indicates potential network packet loss or server downtime.
